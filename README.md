# Портфолио на React

Адаптивный одностраничный сайт-портфолио, собранный на React + Vite. Главный экран содержит карточки проектов, каждая ведёт к подробному описанию и ссылкам на демо/репозиторий. Макет оптимизирован под десктоп, планшеты и мобильные устройства.

## Что нужно на macOS

1. **Homebrew (если ещё нет):**
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. **Node.js (LTS) и npm через Homebrew:**
   ```bash
   brew install node
   node -v
   npm -v
   ```
3. **(Опционально) pnpm или yarn, если предпочитаете альтернативные менеджеры пакетов.**

## Локальный запуск

```bash
cd /Users/gray_advantage/Programming/JS/Spec/Front-Back-3-4
npm install
npm run dev
```

После запуска Vite откроет проект на `http://localhost:5173`. Если вкладка не открылась автоматически, перейдите по адресу вручную.

## Что показать преподавателю

1. Версии Node.js и npm (`node -v`, `npm -v`).
2. Логи установки зависимостей (`npm install`).
3. Работу дев-сервера (`npm run dev`) и открытый сайт в браузере.
4. Адаптивность: уменьшите ширину окна и покажите перестроение карточек/меню.
5. Ссылки из карточек (демо/репозиторий) для подтверждения функциональности.
